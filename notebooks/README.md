# Notebooks Folder

This folder contains all Google Colab notebooks used for the ISOM 835 Transportation Accident Prediction Project.

Each notebook represents a key stage in the machine learning workflow:

- **01_EDA.ipynb** — Exploratory Data Analysis  
- **02_Preprocessing.ipynb** — Data cleaning, handling missing values, encoding, scaling  
- **03_Modeling.ipynb** — Machine learning model training and evaluation  
- **04_Final_Models.ipynb** — Hyperparameter tuning and final model selection

The notebooks include:
- Clear markdown explanations  
- Visualizations saved in the /visualizations folder  
- Reproducible code following best practices

All analysis was completed using Python and Google Colab.
